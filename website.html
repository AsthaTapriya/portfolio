Project Details

A Smart City Complaint and Grievance Portal built using HTML, CSS, JavaScript, Java Spring Boot, MySQL, and Python Flask. It allows citizens to file civic complaints, track their resolution in real time, and lets officers and admins manage the entire complaint lifecycle.

    Smart City Complaint Portal

  • Category: Full Stack Web Development
  • Designed By: Astha Tapriya
  • Language Used: HTML, CSS, JavaScript, Java, Python, SQL
  • Project Date: April 2026
  • Page URL: https://smartcity-portal.netlify.app

What's inside the Web Application?

SmartCity Portal is a full-stack civic complaint management system with three login roles — citizen, officer, and admin. Citizens file complaints with location and priority, with AI-powered department detection built in Python Flask. Each complaint has a real-time status timeline trackable by Complaint ID. Officers manage assigned complaints, while the admin dashboard shows live analytics, manages users, and posts announcements. The backend uses Java Spring Boot REST APIs connected to a MySQL database.
